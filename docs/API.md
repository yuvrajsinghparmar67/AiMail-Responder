# API Documentation

Base URL: `http://localhost:8000` (local) ¬∑ all routes below are prefixed with `/api` except `/health`.

Interactive docs are also auto-generated by FastAPI at `/docs` (Swagger UI) and `/redoc` once the backend is running.

## Authentication

Every protected endpoint expects:

```
Authorization: Bearer <access_token>
```

Tokens are issued by `/api/auth/register` and `/api/auth/login`, expire after `JWT_ACCESS_TOKEN_EXPIRE_MINUTES` (default 7 days), and encode the user's id as `sub`.

## Auth

### `POST /api/auth/register`
Rate limited: `RATE_LIMIT_AUTH` (default 10/minute).

**Body**
```json
{ "email": "jane@company.com", "password": "at-least-8-chars", "full_name": "Jane Doe" }
```

**201 Response**
```json
{
  "access_token": "eyJ...",
  "token_type": "bearer",
  "user": { "id": 1, "email": "jane@company.com", "full_name": "Jane Doe", "created_at": "2026-07-19T12:00:00Z" }
}
```
`409` if the email is already registered.

### `POST /api/auth/login`
Rate limited: `RATE_LIMIT_AUTH`. Body: `{ "email", "password" }`. Same response shape as register. `401` on bad credentials.

### `GET /api/auth/me`
Protected. Returns the current user's profile.

## Emails

### `POST /api/emails/generate`
Rate limited: `RATE_LIMIT_GENERATE` (default 20/minute). **Streams via Server-Sent Events** (`Content-Type: text/event-stream`) ‚Äî not a plain JSON response.

**Body**
```json
{
  "incoming_text": "Hi, when will my order ship?",
  "tone": "professional",
  "length": "medium",
  "language": "english",
  "custom_instructions": "Mention the order shipped yesterday"
}
```
`tone`: `professional | friendly | formal | casual | apologetic | sales | customer_support | executive`
`length`: `short | medium | long`
`language`: `english | hindi | spanish | french | german`

**Stream frames** (each is `data: {json}\n\n`):
```
data: {"type": "token", "token": "Hi"}
data: {"type": "token", "token": " there,"}
...
data: {"type": "done", "generation": { "id": 42, "incoming_text": "...", "tone": "professional", "length": "medium", "language": "english", "generated_reply": "Hi there, ...", "token_usage": 118, "created_at": "..." }}
```
On failure: `data: {"type": "error", "message": "..."}`.

The client must read the response body as a stream (native `EventSource` can't send a POST body or auth headers) ‚Äî see `frontend/lib/api.ts`'s `streamSSE()` for a reference implementation.

### `POST /api/emails/regenerate`
Same streaming contract as `/generate`. Body: `{ "generation_id": 42, "tone"?, "length"?, "language"?, "custom_instructions"? }` ‚Äî omitted fields fall back to the original generation's values. Creates a **new** `EmailGeneration` row (history-preserving, doesn't overwrite the original). `404` if `generation_id` doesn't belong to the caller.

### `POST /api/emails/variations`
Rate limited: `RATE_LIMIT_GENERATE`. **Not streamed** ‚Äî see `llm_service.py` docstring for why. Body: `{ "incoming_text", "tone", "length", "language", "count": 2-4 }`.

**200 Response**
```json
{ "variations": [{ "generated_reply": "...", "token_usage": 96 }, { "generated_reply": "...", "token_usage": 102 }] }
```
Not persisted to history ‚Äî exploratory by design.

### `GET /api/emails/history`
Query params: `search?`, `limit` (1-100, default 20), `offset` (default 0). Returns `EmailGenerationResponse[]`, newest first.

## Drafts

### `POST /api/drafts`
Body: `{ "title", "content", "source_generation_id"? }`. Returns the created `DraftResponse`, `201`.

### `GET /api/drafts`
Query params: `search?`, `favorites_only` (default false), `limit`, `offset`. Returns `DraftResponse[]`, newest-updated first.

### `PUT /api/drafts/{draft_id}`
Body: any subset of `{ "title", "content" }` (partial update). `404` if not found/not owned.

### `PATCH /api/drafts/{draft_id}/favorite`
No body ‚Äî flips `is_favorite`. Returns the updated draft.

### `DELETE /api/drafts/{draft_id}`
`204` on success.

---

## Templates

### `GET /api/templates`
Query param: `category?`. Returns `TemplateResponse[]`.

### `GET /api/templates/{template_id}`
Single template, `404` if not found/not owned.

### `POST /api/templates`
Body: `{ "name", "category", "prompt_text" }`. `category` is free text ‚Äî see `schemas/template.py`'s `SUGGESTED_CATEGORIES` for the suggested set (Customer Support, HR, Sales, Marketing, Recruitment, Internal Team).

### `PUT /api/templates/{template_id}`
Partial update, same shape as create.

### `DELETE /api/templates/{template_id}`
`204` on success.

## Analytics

### `GET /api/analytics/summary`
```json
{
  "total_emails_generated": 128,
  "total_drafts_saved": 34,
  "total_token_usage": 41200,
  "daily_usage": [{ "date": "2026-07-06", "count": 0 }, ..., { "date": "2026-07-19", "count": 9 }]
}
```
`daily_usage` always covers exactly 14 days and is zero-filled for days with no activity, so charts never render with gaps.

## System

### `GET /health`
Unauthenticated. `{ "status": "ok", "service": "AI Email Responder", "environment": "development" }` ‚Äî used by Docker's healthcheck.

## Error shape

All errors follow FastAPI's default: `{ "detail": "human-readable message" }` with the relevant HTTP status code (400/401/404/409/429/500).
